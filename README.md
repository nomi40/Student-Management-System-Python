# Student Management System (Python)

A simple command-line based Student Management System developed in Python.
This project allows users to manage student records efficiently through different operations such as adding, viewing, searching, updating, and deleting student data.

---

## Features

✔ Add Student Records
✔ View All Students
✔ Search Student by Roll Number
✔ Update Student Information
✔ Delete Student Records
✔ Automatic Grade Calculation
✔ Percentage Calculation

---

## Technologies Used

* Python

---

## How the System Works

The system stores:

* Student Name
* Roll Number
* Total Marks
* Grades

The user interacts with the program through a menu-driven interface in the terminal.

---

## Menu Options

1. Add Student
2. View Students
3. Search Student
4. Update Student
5. Delete Student
6. Exit

## How to Run the Project

### Step 1:

Install Python on your system.

### Step 2:

Clone this repository:

```bash
git clone https://github.com/nomi40/Student-Management-System-Python.git
```

### Step 3:

Open the project folder.

### Step 4:

Run the Python file:

```bash
python main.py
```
